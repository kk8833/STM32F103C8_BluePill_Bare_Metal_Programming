# STM32 Bare-Metal Learning Repository

## Overview

This repository documents my hands-on bare-metal embedded systems learning journey using the STM32F103C8T6 (Blue Pill) microcontroller.

The goal of this repository is to understand the internal working of STM32 peripherals through direct register-level programming and memory-mapped I/O without relying heavily on high-level abstraction libraries.

## Topics Covered

This repository progressively covers:

- GPIO
- Timers
- PWM
- USART / UART
- SPI
- I2C
- ADC
- CAN
- Interrupts
- NVIC
- SysTick
- DMA
- RCC Clock Configuration

## Programming Approach

- Bare-metal programming
- Direct register manipulation
- Memory-mapped peripheral access
- Datasheet-oriented development
- Embedded C programming
## Hardware Used

- STM32F103C8T6 Blue Pill
- ST-Link V2
- Breadboard and peripheral modules

## Development Environment

- STM32CubeIDE
- ARM GCC Toolchain
- Embedded C
